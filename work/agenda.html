Gerador de Cronogramas

Categoria

Aplicação de Linha de Comando (CLI)
Ano

2023
Tecnologias

Java, JDBC, MySQL

Este projeto é um sistema de linha de comando (CLI) desenvolvido em Java que cria um cronograma personalizado de estudos. Com base nas matérias selecionadas pelo aluno, a aplicação organiza e exibe um calendário de trabalhos, provas e outras atividades, ajudando na gestão do tempo e na organização acadêmica.
